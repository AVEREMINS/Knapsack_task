# Задача о рюкзаке
## Что надо
Есть набор предметов. У каждого предмета есть: ценность и вес
Ещё есть максимальная вместимость рюкзака.
Нужно выбрать такой набор предметов, чтобы:
1!)суммарный вес не превышал лимит
2)суммарная ценность была как можно больше
То есть надо набрать в рюкзак самые выгодные вещи, но не перегрузить его

Глупо перебить всё - не для нас, а локальным поиском через Хэммминга - то что было на лекции.

Идея такая:
1. Сначала строится стартовое решение жадным способом по выгодности `value/weight`
2. Потом решение улучшается локальным поиском в окрестности Хэмминга
3. Дополнительно используются случайные старты,чтобы не застрять в одном плохом локальном решении

1. Жадный старт
Сначала предметы сортируются по выгодности `value / weight`. Потом в рюкзак по порядку кладётся всё, что помещается.Это не всегда даёт лучший ответ, но даёт хороший старт для дальнейшего улучшения.
2. One bit move - это изменение одного бита в маске решения:
- либо добавить один предмет
- либо убрать один предмет
То есть решение чуть-чуть меняется и проверяется, стало ли лучше.
3. Swap move - это замена:
- один предмет убрать
- другой предмет добавить
4. Local search - программа многократно применяет:
- one bit move
- swap move
Пока решение улучшается.
5. Random start -  то о чем говорилось выше про стоп в плохом месте
Кроме жадного старта используются ещё и случайные старты.
